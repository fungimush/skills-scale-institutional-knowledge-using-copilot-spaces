## How to Check Pull Requests

You can check open pull requests in several ways:

1. **Via GitHub Web Interface**:
   - Navigate to https://github.com/fungimush/skills-scale-institutional-knowledge-using-copilot-spaces/pulls

2. **Via Copilot Space**:
   - Use the prompt "check open pull requests" in your OctoAcme Project Management Hub Copilot Space

3. **Via Mission Control**:
   - Track coding agent progress at GitHub Mission Control for agent tasks

## Pull Request Workflow

1. **Creation**: Pull requests are created either manually or via GitHub Copilot coding agent
2. **Review**: Assigned reviewers (@fungimush) evaluate the changes
3. **Approval**: Reviewers approve changes after validation
4. **Merge**: Once approved, the PR is merged to the main branch
5. **Closure**: The related issue is closed upon successful merge
